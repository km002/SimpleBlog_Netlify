解決に至った対処法を以下に記載します。

1. ターミナルで
```
hexo clean
hexo deploy -g
```
を入力
2. ブラウザのキャッシュを削除
3. 自分のWebサイトでCSSが適用されているか確認
4. 1～3を繰り返す

以上です。
